# MP3Builder

[![Netlify Status](https://api.netlify.com/api/v1/badges/1d018998-c0c0-4f7f-b4f1-bb07ba13086e/deploy-status)](https://app.netlify.com/sites/mp3builder/deploys) 

MP3 builder is a tool for extracting MP3 files from YouTube videos and downloading the files 
with custom MP3 metadata (title, artist name, tags, etc.).

## Features

- Download high-quality audio files from YouTube
- Customize MP3 tags when downloading from YouTube
- Zero ads or trackers
- Files remain on servers for 2 hours before automatic deletion
